Condicionales

En programación, las condicionales son como decisiones en un camino. Imagina que tu programa puede elegir entre diferentes opciones según ciertas condiciones. Esto permite que nuestro codigo tenga intiligencia y se adapte a una situacion en concreto.

Si o if

El if en programación decide ejecutar un bloque de código si una condición es verdadera; de lo contrario, se omite. Esta estructura condicional es fundamental para adaptar el comportamiento del programa según condiciones específicas, permitiendo tomar decisiones y ejecutar acciones selectivamente.

Caso contrario o else

El else en programación se utiliza en conjunto con el if. Si la condición del if es falsa, se ejecuta el bloque de código dentro del "else". Es una forma de proporcionar una alternativa cuando la condición inicial no se cumple. Esto permite manejar múltiples escenarios y tomar decisiones más completas en el código. 

                        
                            num1 = int()
                            num2 = int()

                            if num1 < num2:
                                print("El numero 1 es mayor al numero 2")
                            else:
                                print("El numero 2 es mayor al numero 1")

                            # Codigo de python del else

Si no o else if

El else if en programación es una extensión del else. Funciona como un if secundario con su propia condición. Si la condición del else if es verdadera, se ejecuta su bloque de código; de lo contrario, se pasa al siguiente else if o al else si no hay más condiciones. Esto permite gestionar múltiples escenarios de manera estructurada en el código. 

                    
                        Edad = int()

                        # En python el "else if" se escribe como "elif"
                        if edad <= 17:
                            print("Eres menor de edad")
                        elif edad > 18:
                            print("Eres mayor de edad")
                        else:
                            print("Tu edad es incorrecta")
