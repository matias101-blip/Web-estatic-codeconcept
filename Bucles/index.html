Bucles

Los bucles en programación son como repeticiones automáticas de tareas. Imagina que tienes una acción que quieres hacer varias veces sin tener que escribir el mismo código una y otra vez. Los bucles te permiten hacer eso de manera eficiente. Funcionan como ciclos, donde una acción se repite hasta que se cumple una condición específica. Los bucles son geniales para procesar listas de cosas o realizar acciones repetitivas sin tener que duplicar el código. En resumen, son como "haz esto una y otra vez hasta que te diga que pares".

Arreglos

Los arreglos en programación son listas que permiten almacenar y organizar datos, y cada elemento tiene un índice que indica su posición en el arreglo. Al trabajar con bucles, puedes utilizar estos índices para recorrer y manipular cada elemento de manera secuencial. La combinación de arreglos con bucles facilita la iteración a través de los elementos, realizando acciones específicas para cada uno mediante el uso de sus índices. Esto hace que trabajar con conjuntos de datos sea más eficiente y flexible en el código. 

                    
                        Lista = ['Pollo','atun','Pescado',]
La pocision son: "Pollo" = 0, "Atun" = 1, "Pescado" = 2

Mientras o While

El bucle "while" en programación es una estructura que repite un bloque de código mientras una condición específica sea verdadera. Es como una instrucción "haz esto mientras algo sea cierto". La condición se evalúa antes de cada repetición, y el bucle continúa hasta que la condición se vuelva falsa. Es útil cuando no se sabe cuántas veces se debe repetir una tarea, ya que depende de una condición en tiempo de ejecución. 

                        
                            arreglo = [1,10,2,4]
                            SeBusca = 2
                            encontrado = False
                            i = 0
                            while i < arreglo:
                                numero = arreglo[i]
                                if numero == SeBusca:
                                    encontrado = True
                            if encontrado:
                                print(f"Se encontro el perdido, esta en la posición {i}")
                            else:
                                print("Nose encontro lo que busca")
Este codigo busca un numero en un arreglo de numeros, este bucle while solo se detiene cuando i ya no sea mayor al arreglo.

Para o For

El bucle "for" en programación es ideal para repeticiones controladas y consta de tres partes: inicialización, condición y actualización. Esta estructura es eficiente para iterar a través de arreglos, ya que puedes utilizar el índice del bucle para acceder y manipular cada elemento de manera sistemática. Su sintaxis simplificada facilita la gestión de repeticiones con un enfoque claro en el rango y la iteración. 

                        
                            arreglo = [2,3,4,5]
                            suma = 0
                            for i in arreglo:
                                suma = i + suma
                            print(suma)
En este bucle se suma todos los numeros del arreglo, en este caso i es el "iterador" como este crecera conforme el bucle cresca podra iterar en el arreglo.
